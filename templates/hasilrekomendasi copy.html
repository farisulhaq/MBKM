Persamaan : Definisi :
Top(lu,N) = {lu(1),...,lu(N)} N = Jumlah Rekomendasi
Precision(Testu,lu,N) = |Top(lu,N)∩Testu|N u = Target User
Recall(Testu,lu,N) = |Top(lu,N)∩Testu|NTestu lu = Data Rekomendasi Film User u
F1-Score(Testu,lu,N) = 2*Recall(Testu,lu,N)*Precision(Testu,lu,N) Recall(Testu,lu,N)+Precision(Testu,lu,N) Top(lu,N) = Top N Rekomendasi Film untuk user u
DCG(Testu,lu,N) = ∑n=1N1log2(1+n)*Π(lu(n)∈Testu) Testu = Data Test (Ground-truth) dari user u
IDCG(N) = ∑n=1N1log2(1+n) Π(lu(n)∈Testu) = Kondisi dimana ketika True bernilai 1 sedangkan False bernilai 0
NDCG(Testu,N) = DCG(Testu,N)IDCG(N)
